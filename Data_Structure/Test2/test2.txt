설명
스택(Stack) 자료구조를 활용한 비밀 메시지 관리 프로그램이다. 프로그램은 메뉴 번호를 입력받아 아래 6가지 기능을 반복 수행한다.

1번 : 메시지를 입력받아 스택에 저장한다. 메시지는 10자 이내 소문자로 입력한다.
2번 : 스택에 저장된 가장 최근 메시지를 거꾸로 출력하고 삭제한다. (예: hello → olleh)
3번 : 스택에 저장된 가장 최근 메시지를 원래 순서대로 꺼내되, 각 알파벳을 다음 알파벳으로 시프트하여 출력하고 삭제한다. (예: abc → bcd)
4번 : 스택에 저장된 가장 최근 메시지의 길이만큼 *를 출력한다. 스택에서 삭제하지 않는다. (예: hello → *****)
5번 : 현재 스택에 저장된 메시지(단어)의 총 개수를 출력한다.
6번 : 프로그램을 종료한다.

메시지 입력 시 스택의 여유 공간이 부족하면 초기화 여부를 묻는다. y 입력 시 스택을 초기화하고 저장하며, 그 외 입력 시 해당 입력을 무시한다. 2번, 3번, 4번 기능은 스택이 비어있을 경우 "스택이 비어있습니다."를 출력한다.

제한
"//코드 작성" 주석이 표시된 부분에만 코드를 작성하며, 그 외 기존 코드는 수정하지 않는다.

전역으로 선언된 배열 자료구조에 인덱스([ ])를 사용하여 직접 접근하거나 값을 수정하지 않는다.

각 스택의 top의 값을 직접 변경하지 않는다. 단, 참조는 가능하다.

데이터의 삽입, 삭제, 조회 등 모든 조작은 반드시 사전에 정의된 인터페이스 함수(예: `push`, `pop`, `peek` 등)만을 호출하여 수행해야 한다.
